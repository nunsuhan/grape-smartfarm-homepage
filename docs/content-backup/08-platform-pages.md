## app/platform/page.tsx — 플랫폼 메인

**H1:** FarmSense Platform
**부제:** Transforming agricultural data into actionable intelligence from farm production to export markets.
**설명:** FarmSense integrates sensor networks, AI analysis, and blockchain traceability to create a complete agricultural intelligence platform.

### Platform Architecture (6단계 파이프라인)

| title | desc |
|-------|------|
| Farm Sensors | IoT sensor network collecting real-time data |
| Environment Data | Climate, soil, and crop condition monitoring |
| AI Crop Analysis | Machine learning models for growth prediction |
| Quality Intelligence | Yield and quality assessment algorithms |
| Blockchain Traceability | Immutable production records |
| Export Markets | Verified premium produce supply chain |

**설명:** This end-to-end platform connects farm production with global markets through verified data intelligence.

### 사용 대상 (Who Uses FarmSense Platform)

- **Farmers:** Smart farming support
- **Importers (Buyers):** Export supply chain
- **Investors/Research:** Data analytics

## app/platform/benefits/page.tsx — 팜센스 도입 효과

**H1:** 팜센스 도입 효과
**부제:** 데이터 기반 영농이 가져오는 4가지 변화

### benefits 배열

| title | subtitle | desc |
|-------|----------|------|
| 병해 피해 감소 | 조기 예측으로 선제 대응 | 환경 데이터 기반 병해 발생 확률 예측(PMI-DSS)으로 병해가 눈에 보이기 전에 방제합니다. 시범 농가에서 병해 피해 면적이 크게 줄어드는 효과를 확인하고 있습니다. |
| 노동력 절감 | 자동 모니터링으로 현장 방문 감소 | 24시간 자동 모니터링으로 매일 하우스를 확인하지 않아도 이상 상황 발생 시 즉시 알림을 받습니다. 모니터링에 쓰이는 시간을 다른 영농 작업에 활용할 수 있습니다. |
| 품질 향상 | 데이터 기반 관리로 당도·균일도 개선 | 수확 시기 예측과 환경 최적화를 통해 목표 당도(18-22 Brix) 달성률을 높입니다. 연도별 품질 데이터 비교로 해마다 더 나은 품질을 만들어갑니다. |
| 수출 대응 | 추적성 확보로 프리미엄 시장 진입 | 블록체인 기반 생산 이력 기록과 GAP 인증 연계로 수출 바이어가 요구하는 추적성 요건을 충족합니다. 검증된 생산 데이터로 프리미엄 가격을 실현합니다. |
