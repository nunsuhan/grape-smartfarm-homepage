## app/solutions/grow/page.tsx — Grow · 생육 관리

**H1:** Grow · 생육 관리
**부제:** 데이터 기반 작물 생육 모니터링

**설명:**
팜센스 Grow는 온도, 습도, 토양수분, 광량, CO2 등 생육에 영향을 미치는 핵심 환경 데이터를 실시간으로 수집합니다. AI 기반 생육 예측 모델이 현재 환경 조건을 분석하여 작물의 생육 상태를 진단하고, 생육 단계별 맞춤 관리 가이드를 제공합니다.

데이터가 쌓일수록 예측 정확도가 높아지며, 농가 고유의 영농 노하우와 결합하여 더욱 정밀한 관리가 가능해집니다.

### features 배열

| title | desc |
|-------|------|
| 실시간 환경 모니터링 | 온도, 습도, 토양수분, 광량, CO2를 24시간 실시간으로 수집합니다. |
| 생육 단계 자동 판단 | AI가 현재 작물의 생육 단계를 자동으로 분석하고 기록합니다. |
| 최적 환경 조건 알림 | 생육 단계별 최적 환경 조건에서 벗어날 경우 즉시 알림을 발송합니다. |
| 생육 이력 기록 및 분석 | 누적된 생육 데이터를 바탕으로 연도별·시기별 비교 분석을 제공합니다. |

**CTA:** 내 농장 시작하기 → /smartfarm/dashboard

주간·월간 환경 리포트와 이상 기후 알림 기능으로 농가의 환경 관리 부담을 크게 줄여줍니다.

| 이상 기후 알림 | 이상 기후 예보 및 하우스 내 급격한 환경 변화 발생 시 즉시 알림을 발송합니다. |
| 적산온도 계산 | 작물 생육 단계별 적산온도를 자동 계산하여 관리 시점을 안내합니다. |

## app/solutions/protect/page.tsx — Protect · 병해충 예측

**H1:** Protect · 병해충 예측
**부제:** AI 기반 병해충 사전 예방 시스템

**설명:**
팜센스 Protect는 PMI-DSS(병해 예측 의사결정 지원 시스템)를 통해 환경 데이터 기반으로 병해 발생 확률을 사전 예측합니다. 병해가 눈에 보이기 전에 선제적으로 대응할 수 있어 약제 사용량을 줄이고 작물 피해를 최소화합니다.

AI 사진 진단 기능으로 현장에서 즉시 병해충 여부를 확인하고, 방제 시기와 방법을 추천받을 수 있습니다.

### features 배열

| title | desc |
|-------|------|
| 병해 발생 확률 예측 (PMI-DSS) | 환경 데이터를 분석하여 노균병·탄저병 등 주요 병해의 발생 확률을 사전 예측합니다. |
| AI 사진 진단 | 스마트폰으로 촬영한 작물 사진을 AI가 분석하여 병해충 증상을 즉시 진단합니다. |
| 방제 시기 자동 추천 | 병해 위험도와 날씨 예보를 고려하여 최적의 방제 시기를 자동으로 추천합니다. |
| 지역별 병해 발생 현황 | 인근 농가의 병해 발생 현황을 공유하여 사전 경보 체계를 구축합니다. |

**CTA:** 병해 진단하기 → /smartfarm/ai-chat

## app/solutions/trace/page.tsx — Trace · 블록체인 추적

**H1:** Trace · 블록체인 추적
**부제:** 생산 이력 블록체인 기록

**설명:**
팜센스 Trace는 생산 과정의 모든 데이터를 블록체인에 자동으로 기록합니다. 재배 환경, 농약 사용 이력, 수확 품질까지 위변조 불가한 기록으로 농산물의 신뢰성을 증명합니다.

QR코드 하나로 소비자와 바이어가 농장부터 식탁까지의 모든 이력을 확인할 수 있어 프리미엄 가격 실현이 가능합니다.

### features 배열

| title | desc |
|-------|------|
| 생산 데이터 자동 블록체인 기록 | 센서 데이터, 농약 이력, 수확 정보가 자동으로 블록체인에 기록되어 위변조가 불가합니다. |
| QR코드 생성 및 인쇄 | 포장 박스에 부착할 QR코드를 자동 생성하여 소비자가 생산 이력을 쉽게 확인할 수 있습니다. |
| 소비자 추적 페이지 | QR 스캔 시 농가 정보, 재배 환경, 품질 데이터를 보여주는 소비자용 페이지를 제공합니다. |
| 수출 서류 자동 연계 | 블록체인 기록 데이터를 수출 서류(검역증, 원산지 증명)와 자동으로 연계합니다. |

**CTA:** 블록체인 기록 확인 → /smartfarm/dashboard

| GAP 인증 연계 | 농약 관리 기록을 GAP 인증 심사 자료로 바로 활용할 수 있도록 서식에 맞춰 제공합니다. |
