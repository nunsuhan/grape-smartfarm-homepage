#### blog-2: 포도 노균병 환경조건별 감염 메커니즘
- **category:** disease / 병해충 관리
- **summary:** 노균병은 습도와 온도의 복합 조건에서 폭발적으로 발생합니다.
- **tags:** 노균병, 감염조건, PMI, 엽면습윤
- **sources:** EPPO Bulletin - Plasmopara viticola (2024), 원예과학기술지 제42권
- **read_time:** 6분 | **created_at:** 2026-02-05

**핵심 내용 - 환경조건별 감염 확률:**

| 조건 | 감염 확률 |
|------|----------|
| 20~25°C + 습도 85%↑ + 4시간 엽면습윤 | 매우 높음 (24시간 내 감염) |
| 15~20°C + 습도 75%↑ + 6시간 엽면습윤 | 높음 (2~3일 내 감염) |
| 25°C↑ + 습도 70%↓ | 낮음 (건조 조건에서 억제) |
| 10°C↓ | 매우 낮음 (포자 활동 정지) |

**"10-10-24" 법칙:** 기온 10°C 이상 + 강우량 10mm 이상 + 새순 10cm 이상 → 48시간 내 감염
PMI 0.5 이상: 선제 방제 권장 / PMI 0.7 이상: 즉시 살포

#### blog-5: 샤인머스캣 고품질 재배 핵심 기술
- **category:** cultivation / 재배 기술
- **summary:** 당도 18Brix 이상, 균일한 과립, 일소 피해 없는 프리미엄 샤인머스캣
- **tags:** 샤인머스캣, GA처리, 당도, 일소피해
- **sources:** 샤인머스캣 재배 매뉴얼 - 경북농업기술원 (2025)
- **read_time:** 7분 | **created_at:** 2026-01-15

**GA(지베렐린) 처리:**
- 1차 (무핵화): 만개 14일 전~만개 시, GA₃ 25ppm + 스트렙토마이신 200ppm
- 2차 (과립 비대): 만개 10~15일 후, GA₃ 25ppm + 풀메트 5ppm (목표 12~15g)

**당도 18Brix 달성:**
- 환상박피 (착색기), 신초당 1송이, 수확 2주 전 관수 제한, 인산칼륨 엽면시비

**일소 피해 방지:** 35°C 이상 + 직사광선 → 과실 표면 45°C까지 상승 가능, 차광 자재(30~40%) 설치

**지역별 특성:**
| 지역 | 특징 | 주의사항 |
|------|------|---------|
| 경북 상주·김천 | 일교차 크고 당도 유리 | 봄 늦서리 주의 |
| 충남 천안·공주 | 안정적 재배 환경 | 여름 고온 일소 관리 |
| 전남 나주·영암 | 온난 기후, 조기 출하 유리 | 태풍 피해 대비 |
