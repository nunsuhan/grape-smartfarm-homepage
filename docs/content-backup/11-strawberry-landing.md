## 섹션 3: 핵심 기능 3가지 (features 배열)

### 기능 1: 생육단계별 양액 자동 분석 (Nutrient AI)

**bullets:**
- 설향 기준 EC 0.6~1.8 dS/m, 5단계 자동 판정
- 배액 EC 분석으로 양분 흡수율 실시간 확인
- 품종별 보정 — 설향·죽향·금실·베리스타
- 팁번·칼슘결핍·철결핍 등 생리장해 자동 경고

**highlights:**
| value | label |
|-------|-------|
| EC 0.6 → 1.8 | 정식기 → 저온단일기 |
| 최대 30% | 품종별 EC 차이 자동 반영 |

**source:** 농촌진흥청(RDA) 딸기 배양액 기준 + 전남농업기술원 실증

---

### 기능 2: 물리 모델 기반 환기 최적화 (Climate AI)

**bullets:**
- Kittas(1997) 자연환기 모델 적용
- 한국 단동/연동 비닐하우스 실측 파라미터 (Cd 0.42, Cw 0.12)
- 생육단계 × 계절별 최적 환기 횟수 자동 계산
- 권장 환기창 개폐율 실시간 제안

**highlights:**
| value | label |
|-------|-------|
| 1~30 ACH | 육묘기 ↔ 착과기 자동 전환 |
| 최대 5배 | 풍향별 환기율 차이 자동 보정 |

**source:** 서울대 여욱현 박사논문(2021) 한국 비닐하우스 CFD 검증

---

### 기능 3: 열화상 카메라로 보이지 않는 문제를 봅니다 (FLIR Thermal) 🏆 서비스 유일

**bullets:**
- FLIR One Pro 스마트폰 부착형으로 간편 촬영
- 수분 스트레스, 뿌리 과열, 환기 사각지대 자동 감지
- CWSI(작물수분스트레스지수) 실시간 산출
- 야간 냉해 위험 조기 경고

**highlights:**
| value | label |
|-------|-------|
| +2°C | 잎 온도 상승 = 수분 스트레스 신호 |
| CWSI 0.55+ | 즉시 관개 필요 기준 |

**source:** Celiktopuz et al. 2023 딸기 CWSI 실측

**badge:** 2026.03 기준, 한국 딸기 스마트팜 서비스 중 열화상 통합 서비스 유일
