# 내비게이션 및 링크 전수 검사 매뉴얼 (Navigation & Link Inspection)

이 스킬은 FarmSense 웹사이트의 모든 페이지와 기능적 요소들이 끊김 없이 연결되어 있는지 확인하는 QA(Quality Assurance) 절차입니다. [The Auditor] 에이전트는 배포 전 반드시 이 절차를 수행해야 합니다.

## 1. 글로벌 내비게이션(GNB) 및 푸터(Footer) 검증
새로운 페이지가 생성될 때마다 사이트 전역 메뉴에 해당 링크가 올바르게 반영되었는지 확인합니다.

- **체크리스트**:
  - [ ] 데스크탑 메뉴(`navbar.tsx`)의 드롭다운 목록에 신규 페이지가 포함되었는가?
  - [ ] 모바일 사이드바/햄버거 메뉴에 신규 페이지가 동기화되었는가?
  - [ ] 푸터(`footer.tsx`)의 사이트맵 섹션에 주요 링크가 업데이트되었는가?
  - [ ] 현재 보고 있는 페이지의 메뉴 아이템이 'Active' 상태(색상 변경 등)로 표시되는가?

## 2. 기능적 링크 및 버튼 전수 검사 (Robot Scan)
단순 페이지 이동뿐만 아니라, 기능적 버튼들의 작동 여부를 검수합니다.

- **체크리스트**:
  - [ ] 모든 '자세히 보기', '바로가기' 버튼 클릭 시 404 에러 페이지가 뜨지 않는가?
  - [ ] 외부 링크(Target="_blank")가 새 탭에서 올바르게 열리는가?
  - [ ] 앵커 링크(예: `/#problem`)가 해당 섹션으로 정확히 스크롤 이동하는가?
  - [ ] '뒤로 가기' 버튼이 의도한 상위 페이지로 이동하는가?

## 3. 모바일 반응형 메뉴 동작 검증 (Mobile Responsiveness)
모바일 환경(뷰포트 너비 768px 미만)에서의 사용자 경험을 최우선으로 점검합니다.

- **체크리스트**:
  - [ ] 햄버거 버튼 클릭 시 메뉴 오버레이가 부드럽게 열리는가?
  - [ ] 서브 메뉴(Dropdown)가 터치 시 정상적으로 펼쳐지고 접히는가?
  - [ ] 메뉴 항목 클릭 시 오버레이가 자동으로 닫히며 해당 페이지로 이동하는가?
  - [ ] 메뉴가 화면 밖으로 잘리거나 스크롤이 불가능한 영역은 없는가?

## 4. 링크 유효성 유지 (Link Rot Prevention)
- 기존 페이지의 경로(URL)를 변경할 경우, 반드시 `next.config.js` 등에서 301 리다이렉트 처리를 하거나, 기존 링크를 모두 찾아 수정해야 합니다.
